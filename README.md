This repository for practicing array from Scratch. RoadMap for to solve Array Problems.
    1. Basics + Traversal
          *What to study: Memory layout of arrays, time complexity of array operations, and standard traversal techniques (forward, backward, jumping).
          *What to practice: In-place modifications, swapping elements, and simple counting operations.
    2.  Prefix Sum
          *What to study: The concept of running totals, range sum queries, and how storing previous sums reduces time complexity from O(n^2) to O(n).
          *What to practice: Finding equilibrium indices, calculating sums of specific subarrays, and optimizing nested loops.
